Гармонизация данных

Исходные показатели Росстата приводятся к сопоставимой форме: метрика определяется парой «код показателя × подкатегория», ключ региона — ОКАТО, при дублях по изданиям берётся свежайшее. Абсолютные величины пересчитываются на душу населения; доли, темпы и индексы уже сопоставимы. Для аналитики берутся только показатели уровня «Регион».

Курируемое ядро метрик

Из тысяч показателей отобрано курируемое ядро ({% if profile %}{{ profile.metrics.core }}{% else %}34{% endif %} метрик) по покрытию, релевантности и неизбыточности, распределённое по шести доменам: экономика, доходы, демография, рынок труда, инфраструктура, здоровье и образование. Значения стандартизируются по году (z-оценка).

Типология

По годам строится кластеризация регионов{% if profile and profile.clustering %} (k={{ profile.clustering.0.k }}){% else %} (k=3){% endif %}; метки кластеров согласованы во времени, так что один и тот же тип сопоставим между годами. {% if profile and profile.clustering %}Среднее качество разбиения (silhouette) ≈ {{ profile.clustering.0.silhouette|floatformat:2 }}; устойчивость типа год к году ≈ {{ profile.clustering.0.stability_pct }}%. {% endif %}Для каждого региона считается расстояние до центра своего типа — мера типичности (насколько регион характерен для типа, а не вероятность смены типа). Принадлежность к типу объясняется через SHAP — это объяснение решения классификатора, не причинность.

Индекс развития

Доменные баллы сводятся в композитный индекс по {% if profile %}{{ profile.index.schemes }} схемам{% else %}трём схемам{% endif %} весов (равные, PCA, экспертные) и нормируются в диапазон 0–100 по всему окну, чтобы рост региона во времени был виден. Устойчивость рейтинга к выбору весов проверяется ранговой корреляцией.

Переходы и траектории

Прослеживается движение регионов между типами год к году и классифицируется тип траектории (устойчивый, сходящийся, расходящийся, волатильный и др.). Изменение индекса показывается как разложение по доменам — это арифметика баллов, без утверждений о причинах.

Честность выводов

Данные — короткие годовые ряды, поэтому методы подбираются под это ограничение, а результаты сопровождаются метриками качества. Формулировки избегают причинности там, где методы её не устанавливают.
