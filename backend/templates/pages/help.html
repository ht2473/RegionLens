{% trans "Разделы" %}

  • {% trans "Карта" %} — {% trans "регионы, раскрашенные по типу или индексу; ползунок выбирает год." %}
  • {% trans "Показатели" %} — {% trans "каталог метрик с поиском: карта значений, таблица и временной ряд по выбранному показателю." %}
  • {% trans "Регионы" %} — {% trans "список субъектов и переход к дашборду конкретного региона." %}
  • {% trans "Рейтинг" %} — {% trans "ранжирование регионов по индексу развития за выбранный год; отдельная вкладка — устойчивость ранга к выбору весов." %}
  • {% trans "Лаборатория индекса" %} — {% trans "чувствительность рейтинга к схеме весов: согласованность схем и коридор ранга региона." %}
  • {% trans "Конвергенция" %} — {% trans "сближаются ли регионы во времени: σ-сходимость (разброс) и β-сходимость (мобильность)." %}
  • {% trans "Неравенство" %} — {% trans "разброс регионов по выбранному показателю и его динамика по годам." %}
  • {% trans "Типология" %} — {% trans "обзор типов регионов и их состава." %}
  • {% trans "Сравнение" %} — {% trans "сопоставление 2–3 регионов по доменам (gap-анализ)." %}
  • {% trans "Аномалии" %} — {% trans "регионы с нетипичными значениями и структурными сдвигами." %}
  • {% trans "Корреляции" %} — {% trans "совместное движение пар показателей по регионам (связь, не причинность)." %}

{% trans "Поиск" %}

{% blocktrans %}Иконка поиска в шапке (или сочетание Ctrl+K) открывает быстрый поиск по регионам, показателям и страницам. В разделах «Неравенство» и «Корреляции» показатель выбирается через поле с поиском по названию.{% endblocktrans %}

{% trans "Обозначения" %}

  • {% trans "Индекс развития" %} — {% trans "композитный балл 0–100 по шести доменам; нормирован по всему окну, чтобы рост региона во времени был сопоставим." %}
  • {% trans "Схемы весов" %} — {% trans "равные, PCA и экспертные; рейтинг приводится для каждой, а его устойчивость проверяется ранговой корреляцией." %}
  • {% trans "Тип (кластер)" %} — {% trans "группа похожих регионов за год; метки согласованы по годам." %}
  • {% trans "Типичность" %} — {% trans "насколько регион характерен для своего типа (не прогноз смены типа)." %}
  • {% trans "SHAP" %} — {% trans "вклад метрик в отнесение региона к типу (объяснение решения классификатора, не причинность)." %}
  • {% trans "σ-сходимость" %} — {% trans "сокращается ли разброс между регионами во времени (коэффициент вариации, индекс Джини, отношение P90/P10)." %}
  • {% trans "β-сходимость" %} — {% trans "догоняют ли изначально отстающие регионы; для относительного индекса трактуется как мобильность, а не абсолютный рост." %}
  • {% trans "Корреляция" %} — {% trans "мера совместного движения показателей; высокая связь не означает причинности." %}

{% trans "Учётная запись" %}

{% trans "Все разделы аналитики открыты без регистрации. Вход нужен только для личного кабинета: сохранённые виды, история экспортов и профиль. Тема оформления (светлая или тёмная) переключается в шапке." %}
