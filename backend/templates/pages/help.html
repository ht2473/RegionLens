Помощь по платформе

Справка

Короткое руководство по разделам и обозначениям. Если что-то не работает или непонятно — напишите через обратную связь.

Разделы

  • Карта — регионы, раскрашенные по типу или индексу; ползунок выбирает год.
  • Показатели — каталог метрик с поиском: карта значений, таблица и временной ряд по выбранному показателю.
  • Регионы — список субъектов и переход к дашборду конкретного региона.
  • Рейтинг — ранжирование регионов по индексу развития за выбранный год; отдельная вкладка — устойчивость ранга к выбору весов.
  • Лаборатория индекса — чувствительность рейтинга к схеме весов: согласованность схем и коридор ранга региона.
  • Конвергенция — сближаются ли регионы во времени: σ-сходимость (разброс) и β-сходимость (мобильность).
  • Неравенство — разброс регионов по выбранному показателю и его динамика по годам.
  • Типология — обзор типов регионов и их состава.
  • Сравнение — сопоставление 2–3 регионов по доменам (gap-анализ).
  • Аномалии — регионы с нетипичными значениями и структурными сдвигами.
  • Корреляции — совместное движение пар показателей по регионам (связь, не причинность).

Поиск

Иконка поиска в шапке (или сочетание Ctrl+K) открывает быстрый поиск по регионам, показателям и страницам. В разделах «Неравенство» и «Корреляции» показатель выбирается через поле с поиском по названию.

Обозначения

  • Индекс развития — композитный балл 0–100 по шести доменам; нормирован по всему окну, чтобы рост региона во времени был сопоставим.
  • Схемы весов — равные, PCA и экспертные; рейтинг приводится для каждой, а его устойчивость проверяется ранговой корреляцией.
  • Тип (кластер) — группа похожих регионов за год; метки согласованы по годам.
  • Типичность — насколько регион характерен для своего типа (не прогноз смены типа).
  • SHAP — вклад метрик в отнесение региона к типу (объяснение решения классификатора, не причинность).
  • σ-сходимость — сокращается ли разброс между регионами во времени (коэффициент вариации, индекс Джини, отношение P90/P10).
  • β-сходимость — догоняют ли изначально отстающие регионы; для относительного индекса трактуется как мобильность, а не абсолютный рост.
  • Корреляция — мера совместного движения показателей; высокая связь не означает причинности.

Учётная запись

Все разделы аналитики открыты без регистрации. Вход нужен только для личного кабинета: сохранённые виды, история экспортов и профиль. Тема оформления (светлая или тёмная) переключается в шапке.
