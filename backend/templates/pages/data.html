Источник и охват

Данные

Платформа работает на открытых данных Росстата, агрегированных порталом «Если быть точным». Персональных данных нет — все показатели агрегированы по регионам.

Источник

  • Росстат / портал «Если быть точным», лицензия CC BY 4.0.
  • Формат — длинная таблица показателей по регионам, федеральным округам и стране.
  • Для аналитики используется только уровень «Регион».

Охват

  • 85 непересекающихся субъектов РФ (агрегаты «с автономным округом» исключены, чтобы не задваивать данные).
  • Полный диапазон рядов — 2001–2025 (для отображения); окно расчёта аналитики — 2010–2024.
  • Ключ региона и стыковка с картой — код ОКАТО.

Качество и воспроизводимость

  • Дедупликация по свежайшему изданию источника; контроль качества как код.
  • Импутация пропусков с предохранителями; доля импутаций контролируется.
  • Сиды зафиксированы — пересборка аналитики детерминирована.

Подробнее о методах расчёта — на странице Методология.
