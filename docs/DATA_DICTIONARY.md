# RegionLens — словарь данных (контрактные таблицы DuckDB)

Справочник по контрактным таблицам аналитического хранилища (файл DuckDB). Это **контракт**, на
который опирается приложение (см. [ARCHITECTURE.md](ARCHITECTURE.md), §1–§2). Таблицы пишет только
офлайн-конвейер; приложение читает их read-only. Колонка «стадия» — владелец таблицы.

Типы указаны в нотации Polars (`Utf8`/`String`, `Int32`/`Int64`, `Float64`, `Boolean`). «Грань» —
ключ уникальности строки.

## Справочники

### `metric_dim` — справочник метрик
Стадия: `etl` (база) → `features` (обогащение). Грань: `metric_id`. Натуральный ключ: `(indicator_code, subsection)`.

| Колонка            | Тип      | Описание                                                                 |
|--------------------|----------|--------------------------------------------------------------------------|
| `metric_id`        | Int      | Суррогатный ключ метрики (1..N), детерминирован между прогонами.          |
| `indicator_code`   | Utf8     | Код показателя Росстата.                                                  |
| `subsection`       | Utf8     | Раздел/подраздел (вторая часть грани метрики).                            |
| `domain`           | Utf8     | Домен: `economy`/`income`/`labor`/`demography`/`health_edu`/`infrastructure`/`excluded`. Добавляет `features`. |
| `value_type`       | Utf8     | Тип величины (влияет на гармонизацию форм). Добавляет `features`.         |
| `higher_is_better` | Boolean  | Направление метрики (используется для знака в индексе). Добавляет `features`. |
| `coverage`         | Float64  | Доля покрытия метрики по окну анализа. Добавляет `features`.              |

### `region_dim` — справочник регионов
Стадия: `etl`. Грань: `okato`.

| Колонка            | Тип            | Описание                                                            |
|--------------------|----------------|---------------------------------------------------------------------|
| `okato`            | Utf8           | Ключ региона (ОКАТО).                                                |
| `oktmo`            | Utf8           | Код ОКТМО.                                                           |
| `region_name`     | Utf8           | Наименование региона.                                               |
| `federal_district` | Utf8 (nullable) | Федеральный округ (из `config/regions.yaml`); незамапленные → null. |
| `included_flag`   | Boolean        | Включён ли вариант региона (отсекает дубль «с/без автономного округа»). |
| `geojson_key`     | Utf8           | Ключ для гео-слоя карты (= `okato`).                                |

## Факты и признаки

### `fact_region` — длинный факт по сырью
Стадия: `etl`. Грань: `(okato, metric_id, year)`. Валидируется pandera (`year ∈ [2001, 2025]`, уникальность грани).

| Колонка     | Тип             | Описание                          |
|-------------|-----------------|-----------------------------------|
| `okato`     | String          | Регион (ОКАТО).                    |
| `metric_id` | Int32           | Метрика (FK → `metric_dim`).       |
| `year`      | Int64           | Год наблюдения.                    |
| `value`     | Float64 (nullable) | Исходное значение показателя.   |
| `source`    | String (nullable)  | Источник наблюдения.            |

### `features_wide` — плотное ядро признаков
Стадия: `features`. Грань: `(okato, metric_id, year)`. Плотная по метрикам ядра × окну (после импутации).

| Колонка            | Тип      | Описание                                                                        |
|--------------------|----------|---------------------------------------------------------------------------------|
| `okato`            | Utf8     | Регион.                                                                          |
| `year`             | Int32    | Год.                                                                             |
| `metric_id`        | Int      | Метрика ядра.                                                                    |
| `value_harmonized` | Float64  | Гармонизованное значение (например, приведение абсолютных величин к подушевым).  |
| `z_value`          | Float64  | z-стандартизация `value_harmonized` по `(metric_id, year)`; знак направления НЕ применён. |
| `is_imputed`       | Boolean  | Признак импутированной (восстановленной) точки.                                  |

| `algo`           | Utf8            | Алгоритм кластеризации (`kmeans`).                              |

## Индекс развития

### `dev_index` — композитный индекс развития
Стадия: `dev_index`. Грань: `(okato, year, weighting_scheme)`.

| Колонка            | Тип     | Описание                                                                 |
|--------------------|---------|--------------------------------------------------------------------------|
| `okato`            | Utf8    | Регион.                                                                  |
| `year`             | Int32   | Год.                                                                     |
| `weighting_scheme` | Utf8    | Схема взвешивания доменов (одна из трёх).                                 |
| `total_score`      | Float64 | Композитный балл развития.                                               |
| `economy`          | Float64 | Доменный балл «экономика».                                               |
| `income`           | Float64 | Доменный балл «доходы».                                                  |
| `labor`            | Float64 | Доменный балл «труд».                                                    |
| `demography`       | Float64 | Доменный балл «демография».                                              |
| `health_edu`       | Float64 | Доменный балл «здоровье и образование».                                  |
| `infrastructure`   | Float64 | Доменный балл «инфраструктура».                                          |

*Доменные столбцы соответствуют активным доменам ядра (`config/indicators.yaml`); домен `excluded` в индекс не входит.*

## Пространственная автокорреляция

### `moran_global` — глобальный Moran's I индекса
Стадия: `spatial`. Грань: `(weighting_scheme, year)`.

| Колонка            | Тип     | Описание                                                                 |
|--------------------|---------|--------------------------------------------------------------------------|
| `weighting_scheme` | Utf8    | Схема весов индекса.                                                     |
| `year`             | Int32   | Год.                                                                     |
| `morans_i`         | Float64 | Глобальный индекс Морана (`>0` — соседние регионы похожи по индексу).     |
| `expected_i`       | Float64 | Ожидание при отсутствии автокорреляции (`-1/(n-1)`).                      |
| `z_score`          | Float64 | Z-оценка отклонения от ожидания.                                         |
| `p_value`          | Float64 | Значимость (перестановочный тест, 999 перестановок, seed=42).            |
| `n_regions`        | Int32   | Число регионов с соседями (эксклавы/острова исключены).                  |

### `moran_local` — локальная автокорреляция (LISA) по регионам
Стадия: `spatial`. Грань: `(weighting_scheme, year, okato)`.

| Колонка            | Тип     | Описание                                                                 |
|--------------------|---------|--------------------------------------------------------------------------|
| `weighting_scheme` | Utf8    | Схема весов индекса.                                                     |
| `year`             | Int32   | Год.                                                                     |
| `okato`            | Utf8    | Регион.                                                                  |
| `local_i`          | Float64 | Локальный индекс Морана (null у регионов без соседей).                   |
| `quadrant`         | Utf8    | Тип кластера при `p<0.05`: `HH`/`LL`/`HL`/`LH`, иначе `ns`.               |
| `p_value`          | Float64 | Перестановочная значимость (null у регионов без соседей).                |
| `n_neighbors`      | Int32   | Число соседей (0 — эксклав/остров).                                      |

*Пространственные веса — смежность субъектов по общей границе (rook), выведенная из geojson; строкан-нормированные. Статистика — библиотека esda (PySAL).*
