# Примеры работы с API RegionLens

## `api_quickstart.ipynb`

Ноутбук с практическими примерами обращения к API из Python: список регионов, рейтинг
индекса развития, индекс со своими весами, декомпозиция изменения индекса, сценарный анализ
и проверка аутентификации по токену.

### Запуск

1. Поднимите сервер RegionLens (например, `python backend/manage.py runserver`).
2. Установите зависимости примера: `pip install requests jupyter`.
3. Откройте ноутбук: `jupyter notebook examples/api_quickstart.ipynb`.
4. В первой ячейке укажите адрес сервера (`BASE`) и, при желании, личный токен (`TOKEN`)
   из личного кабинета → «API-доступ».

Токен не обязателен для чтения открытых данных, но повышает лимит частоты запросов
(120 → 600 в минуту) и идентифицирует запросы как ваши.

Полная интерактивная документация эндпойнтов доступна на работающем сервере по адресу
`/api/v1/docs/` (Swagger UI); машиночитаемая схема OpenAPI — `/api/v1/schema/`.
